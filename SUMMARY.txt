⚡ QUICK START COMMANDS
═══════════════════════════════════════════════════════════════════════════════

# 1. Clone/Download the project
git clone https://github.com/yourusername/surgipredict-pro.git
cd surgipredict-pro

# 2. Setup environment
cp .env.example .env.local
echo "REACT_APP_ANTHROPIC_API_KEY=YOUR_API_KEY_HERE" >> .env.local

# 3. Install dependencies
npm install

# 4. Start development server
npm start

# 5. Open browser
open http://localhost:3000

# 6. For production build
npm run build
vercel --prod

Surgical Parameters Available:
├─ Nasal Bridge: -100 to +100 (Reduction ← → Augmentation)
├─ Nasal Width: -100 to +100 (Narrowing ← → Widening)
├─ Lip Volume: -100 to +100 (Reduction ← → Augmentation)
├─ Jawline: -100 to +100 (Softening ← → Definition)
├─ Cheekbones: -100 to +100 (Reduction ← → Projection)
├─ Skin Tightness: -100 to +100 (Laxity ← → Tension)
└─ Asymmetry Correction: 0 to +100 (None ← → Complete)

API Endpoints (Backend):
├─ POST /api/analyze-image (Image analysis with AI vision)
├─ POST /api/recommendations (Surgical recommendations)
├─ POST /api/compare-techniques (Technique comparison)
├─ POST /api/generate-report (Report generation)
├─ GET  /api/health (Health check)
└─ GET  /api/rate-limit (Rate limiting info)
